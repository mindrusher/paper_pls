# paper_pls API

API-приложение для управления постами.

------------------------------------

## Стек

- Python 3.11
- FastAPI
- SQLAlchemy
- Pydantic
- PostgreSQL
- Redis
- Docker / Docker Compose

------------------------------------

## Функциональность
- Слоистая архитектура:
    - `api` - зависимости и маршруты FastAPI
    - `application` - управление сценариями
    - `domain` - бизнес-логика
    - `infrastructure` - реализации БД
    - `scripts` - технические утилиты

- Точки входа:
  - получение поста по ID
  - получение всех постов
  - создание поста
  - обновление поста
  - удаление поста

------------------------------------

## Требования
- Docker
- Docker Compose

------------------------------------


## Установка и запуск

## 1. Клонировать репозиторий

```bash
git clone <https://github.com/mindrusher/paper_pls.git>
cd paper_pls
```

## 2. Собрать и запустить контейнеры

```bash
docker-compose up --build
```

## 2. Документация API (Swagger UI)

http://localhost:8000/docs

## 3. Все запросы требуют API-ключа в заголовке


## 4. Остановка контейнеров

```bash
docker-compose down -v
docker-compose up --build
```
